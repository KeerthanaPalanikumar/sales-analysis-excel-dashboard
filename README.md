# 📊 Sales Analysis Dashboard (Excel)

## 📌 Overview

Interactive **Sales Dashboard** built in Excel to analyze performance by product, year, and region using dynamic visuals and filters.


## 🛠️ Tools & Technologies

* **Excel**
* **Formulas:** `SUMIFS`, `COUNTIFS`, `MAX`, `UNIQUE`, `INDEX-MATCH`
* **Interactivity:** Combo Box, Data Validation
* **Charts:** Stacked Column, Donut, Map, Sparklines
* **Other:** Conditional Formatting


## 📈 Features

* Dynamic filtering (Year & Product)
* KPI tracking (Sales, Max values)
* Trend & comparison analysis
* Region-wise insights


## 🚀 How to Use

1. Open the Excel file
2. Go to **DASHBOARD**
3. Select filters using Combo Box
4. View updated insights
